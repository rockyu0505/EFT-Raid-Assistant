EFT Raid Assistant 0.5.0 使用说明
=================================

一、这是什么
------------

EFT Raid Assistant 是一个面向中文玩家的 Escape from Tarkov 局内辅助工具。

它用于：
- 鼠标悬停物品后 OCR 识别名称并查价。
- 显示 PvE / PvP 本地缓存价格。
- 用颜色提示物品价值区间。
- 在右上角显示最多三条查价浮窗。
- 提供商人补货倒计时提醒功能。

它不会：
- 点击鼠标或移动鼠标。
- 自动购买、出售或整理物品。
- 读取游戏内存。
- 注入游戏进程。
- 修改游戏文件。


二、安装和启动
--------------

1. 请完整解压整个 zip 压缩包。

   推荐路径示例：
   D:\GamesTools\EFT-Raid-Assistant

2. 解压后双击运行：

   EFT Raid Assistant.exe

3. 不需要安装 Python、Miniconda 或 PySide6。

4. 不要只单独复制或发送 exe。

   程序必须和旁边的 _internal、cache、data、assets 等文件夹放在一起运行。

5. 如果 Windows SmartScreen 提示“未知发布者”，可以点击“更多信息”，然后选择“仍要运行”。


三、默认热键
------------

Q   ：局内物品查价
F6  ：识别藏身处升级材料
F8  ：识别商人倒计时
F10 ：设置选中商人的补货提醒

热键可以在 Settings 中修改。


四、局内物品查价
----------------

1. 打开 Tarkov 装备、背包或容器界面。
2. 把鼠标悬停到要查询的物品上。
3. 等游戏显示完整物品名称 tooltip。
4. 保持 Tarkov 是前台窗口，按 Q。
5. 程序会截图鼠标附近的小区域，定位 tooltip，OCR 识别物品名，并查询本地价格缓存。
6. 查价结果会显示在主界面和右上角浮窗。

如果识别结果明显不对，可以在主界面手动输入物品名重新查询。


五、PvE / PvP 价格模式
----------------------

主界面可以手动选择 PvE 或 PvP。

默认是 PvE。

当前版本不再自动 OCR 判断 PvE/PvP，这样可以减少截图和误判。


六、价格颜色
------------

普通物品默认按单格价值显示颜色：

0 - 1 万       白色
1 - 2 万       绿色
2 - 5 万       蓝色
5 - 10 万      紫色
10 - 25 万     金色
25 - 50 万     红色
50 万以上      彩色渐变

枪械会使用迷彩绿色提示。

原因是 Tarkov 中枪械本体通常只按下机匣体积计算，但实际出售价值包含大量配件，单格价值容易误导。


七、OCR 引擎
------------

默认推荐：
RapidOCR only

可选实验项：
RapidOCR v5 only

0.5.0 统一使用 RapidOCR v5，并优化了物品查价、藏身处识别、完整物品名匹配和简繁字符归一化。

如果遇到个别物品识别不稳定，可以在 Settings 中切换 OCR 引擎进行对照。


八、右上角浮窗
--------------

查价后会在右上角显示半透明价格卡片。

默认最多保留三条记录。

到时间后自动淡出。

显示时间可以在 Settings 中调整。


九、商人补货提醒
----------------

商人补货提醒仍属于实验功能。

默认热键：
- F8：识别商人倒计时
- F10：设置选中商人的提醒

主流程建议优先使用局内查价功能。


十、日志和调试文件
------------------

主界面只显示关键结果：
- 查价成功
- 没有匹配物品
- 非 Tarkov 前台窗口
- 当前界面不适合查价

完整运行日志：
debug\latest_run.log

最近一次截图和 OCR 调试图也保存在 debug 文件夹。

如果要反馈识别问题，请把 debug 文件夹和 latest_run.log 一起发给开发者。


十一、常见问题
--------------

1. 按 Q 没反应

   确认程序正在运行，且 Q 没有被其他软件或游戏设置冲突占用。
   也可以在 Settings 中改成其他热键。

2. 提示 Tarkov 不是前台窗口

   重新点击 Tarkov，让游戏成为前台窗口后再按 Q。

3. 识别错物品

   等 tooltip 完整显示后再按热键。
   如果仍然错误，可以手动输入物品名查价，并保留 debug 文件用于排查。

4. 价格模式不对

   在主界面手动选择 PvE 或 PvP。

5. exe 单独复制后无法运行

   这是正常的。请完整解压并保留 _internal 等目录。

十三、0.5.0 主要更新
--------------------

- 新增藏身处升级材料识别与需求记录 MVP。
- 新增藏身处识别热键，默认 F6。
- 新增关闭窗口时最小化到托盘/退出/取消选择。
- 新增应用图标，并用于主窗口、托盘和 exe。
- 手动查价输入框支持自动补全候选物品。
- 统一使用 RapidOCR v5，移除 Tesseract 运行时和旧设置。
- 查价悬浮窗优先显示总参考价，其下显示单格价值，并对高波动物品追加历史价格评估。
